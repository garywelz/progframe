Gödel Numbering (syntactic encoding)

Mathematics Number Theory / Logic Paired proof graph: Gödel First Incompleteness
← Back to Mathematics Database First Incompleteness (proof graph)

Description

Assign a unique natural number to each formula of arithmetic by recursively encoding the syntax tree with prime exponentiation (standard Gödel-style numbering). This algorithm is the algorithm capsule sibling of the logical diagonal construction in the incompleteness proof graph.
