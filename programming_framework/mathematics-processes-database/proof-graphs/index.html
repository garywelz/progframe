Recommendation

Proof graphs should be attached to theorem entries as drill-down views and listed as a distinct proof-level category inside the mathematics database. This keeps theorem dependencies, proof dependencies, and algorithmic procedures distinct while allowing alternate proofs of the same theorem to be compared.
