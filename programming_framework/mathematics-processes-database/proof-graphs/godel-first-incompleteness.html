Gödel First Incompleteness Theorem (informal proof graph)

Diagonalization builds a sentence G that asserts its own unprovability. Structurally this parallels Cantor diagonal arguments: both encode "apply a transformation derived from a listing to obtain an object off the list / off the naive predicate".
